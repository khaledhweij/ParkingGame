# ParkingGame
A basic parking game using Unity3D for a game programming course at the University of Nicosia 

Game instructions:

Press D to start driving the car. The car will start moving straight away and you can increase the speed by pressing â†‘ (up key).
To drive backward, you must press the R key first.
To win the game, the P key must be pressed when the car is parked in the correct place (it is indicated with an arrow on the ground).
The game will restart from the beginning if the car hits anything (e.g. cars, buildings, trees).
The player has 25 seconds to stop the car and win the game otherwise he/she will lose, and the game will restart automatically.
The player has the ability to change the position of the camera, zoom in and out using the mouse.
